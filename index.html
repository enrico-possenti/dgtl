dgtl by noveopiu

dCTL is a framework for collections of digital objects along with methods for access and retrieval, and for organization and maintenance of collections themselves. This software provides a convenient way to organizing information and making it available over the Internet.

The dCTL framework satisfies the physical needs of a DL: * a way for encoding of sources into XML e multimedia resources * a management system for those documents * a repository to store them * an interface to retrieve information The dCTL framework replies to these needs with an integrated suite of modular and extensible set of tools, where each one can get input and produce output indipendently from the other, but if used in conjunction the framework is a powerful Content Management System. Actually dCTL framework is [http://www.tei-c.org/ TEI]-oriented in every component and module for historical-but-still-more-than-valid choices done in the past, but could easily be generalized to handle different schemas with a little effort.
